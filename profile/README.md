# GDG On Campus MCKVIE

Welcome to the official GitHub organization of **GDG On Campus MCKV Institute of Engineering**.

## 🙋‍♀️ About Us
GDG On Campus MCKVIE is a student developer community supported by Google Developer Groups (GDG).  
Our mission is to help students learn modern technologies, collaborate on projects, and grow as developers.

We organize:
- Technical Workshops
- Hackathons
- Study Jams
- Coding Sessions
- Open Source Contributions

## 🌈 How to Contribute
We welcome students and developers to contribute to our projects.

Steps to contribute:
1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a Pull Request
