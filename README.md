# GDG On Campus MCKVIE – Organization Profile

This repository contains the **organization profile README** for the GitHub organization of GDG On Campus MCKV Institute of Engineering.

The content located in:

.github/profile/README.md

is displayed on the main organization profile page.

## About GDG On Campus MCKVIE

GDG On Campus MCKVIE is a student-led developer community supported by Google Developer Groups.  
Our mission is to help students learn modern technologies, collaborate on projects, and grow through workshops, hackathons, and open-source initiatives.

## Purpose of This Repository

This repository is used to:
- Maintain the public GitHub organization profile
- Share community information
- Provide resources and links for members and contributors

For more information about our activities, visit the **organization profile page**.
